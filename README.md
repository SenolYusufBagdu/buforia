# Buforia - AR Credit Card Tracker

A Unity-based augmented reality project that detects credit cards using image recognition and places a 3D cube on top of them in real time.

## Features

* Real-time credit card detection
* 3D object placement on detected cards
* Built using Unity and Vuforia Engine

## Technologies

* Unity
* Vuforia Engine 10.25.4
* C#

## How to Run

1. Clone the repository
2. Open the project with Unity
3. Connect your Android or iOS device
4. Press **Play** or build the project to your device
5. Point the camera at a credit card to see the AR object

# Buforia - AR Kredi Kartı Takip Uygulaması

Unity ile geliştirilmiş bir artırılmış gerçeklik projesidir. Görüntü tanıma teknolojisi kullanarak kredi kartlarını tespit eder ve kartın üzerine gerçek zamanlı olarak bir 3D küp yerleştirir.

## Özellikler

* Gerçek zamanlı kredi kartı tespiti
* Tespit edilen kartın üzerine 3D nesne yerleştirme
* Unity ve Vuforia Engine kullanılarak geliştirildi

## Kullanılan Teknolojiler

* Unity
* Vuforia Engine 10.25.4
* C#

## Nasıl Çalıştırılır

1. Repoyu klonlayın
2. Projeyi Unity ile açın
3. Android veya iOS cihazınızı bağlayın
4. **Play** tuşuna basın veya projeyi cihaza derleyin
5. Kamerayı bir kredi kartına tutarak AR nesnesini görüntüleyin
